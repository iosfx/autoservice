# AutoService MVP v1

## Stack
- Backend: Node.js + TypeScript
- DB: PostgreSQL
- iOS app: SwiftUI (later)
- Admin console: later

## MVP Features (backend first)
- Models: Client, Vehicle, Job, MessageLog, Shop, User
- REST API
- Health endpoint
- Auth (simple JWT)
- Event-driven job status updates
- Messaging engine (mock providers first)

## Non-goals in v1
- Client app
- Payments
- Invoicing
